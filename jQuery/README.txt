Коллекция из 10 практических заданий на jQuery, демонстрирующих работу с DOM, событиями, анимациями, AJAX и jQuery UI.

Небольшой путеводитель: 
 jquery-1:
01-dropdown-menu - Выпадающее меню с обработкой кликов вне области
02-list-filter - Живой поиск по списку с filter()
03-drag-drop - Drag & Drop через jQuery UI
04-sortable-table - Сортировка таблицы по колонкам
05-autocomplete - Автозаполнение с локальной базой городов

jquery-2:
01-api-content - Загрузка и отображение данных из JSONPlaceholder
02-image-gallery - Галерея с увеличением и затемнением фона
03-calculator - Динамический калькулятор формы
04-notifications - Всплывающие уведомления с авто-скрытием
05-chat-sim - Имитация чата с ботом (setTimeout)

Технологии:
- jQuery 3.7.1 (через CDN)
- jQuery UI 1.13.2 (для drag & drop)
- HTML5 / CSS3 (Flexbox, Grid, анимации)
- Нативный JavaScript (для логики)
