Игра "Угадай число" (PHP + Sessions)
Простая браузерная игра с бэкендом на PHP. Загаданное число и история попыток сохраняются между перезагрузками страницы с помощью `$_SESSION`.

Технологии
- PHP 8+
- HTML5 / CSS3
- OpenServer

Функционал
- Генерация случайного числа (1–100) при первом входе
- Проверка догадки на стороне сервера
- История попыток сохраняется
- Кнопка "Сдаться" показывает число и сбрасывает игру
- При победе таблица очищается, загадывается новое число

Как запустить локально
В некоторых случаях потребуется сменить направление слеша в файле ".osp".
1. Установить OSPanel (Open Server Panel)
2. Переместить папку `php-guess-number` в `OSPanel/home/`
 Важно: в этой папке создать папку ".osp", создать файл "project" и вставить это: 
"
[php-guess-number]

http_engine = Apache
php_engine  = php-8.4
web_root    = {base_dir}\public
"

3. Запустить OpenServer и открыть появившийся проект из списка

Все данные обрабатываются на сервере. HTML-атрибуты `min/max` только для UX, реальная валидация происходит в PHP.
